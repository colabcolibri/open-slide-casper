# Themes on slides

If `themes/<id>.md` exists and the slide should follow it, **the theme file overrides defaults in this skill** — palette, typography, padding, Title/Footer are authoritative. Read the theme before applying canvas/type rules elsewhere.

Themes come from `create-theme`. Copy palette and paste-ready Title / Footer / Eyebrow into the slide. Frontmatter `mode: dark` | `light` hints background treatment (e.g. logo variant).
